#Nuxt는 알았는데 Next는 뭐야??
next.js는 Nuxt.js 처럼 react.js에 서버사이드 렌더링 쉽게 도와주는 프레임워크다.

장점 -기본적으로 서버사이드 렌더링을 제공함

- 서버사이드 렌더링의 장점
- 검색엔진 최적화
- 초기 로딩 성능 개선
- 그냥 검색엔진 최적화
- SEO (Search Engine Optimization)
  더욱 빠른 페이지 로드를 위한 코드 스플리팅 자동화
  HMR을 지원하는 웹팩 기반 환경
  Express나 Node.js와 같은 http 서버와 함께 구현 가능
  Babel, Webpack 설정 커스터마이징 가능
